\# 🛡️ CyberShadow Suite



Suite modulare per security testing e analisi log USB/ADB.



\## 🚀 Moduli

\* \*\*CyberShadow Core\*\*: Shell interattiva con database hash auto-apprendente.

\* \*\*VulnShop\*\*: Web app vulnerabile (SQLi, XSS) per test di penetrazione.

\* \*\*LogShadow\*\*: Acquisizione logcat in tempo reale da dispositivi USB.

\* \*\*Crypto\*\*: Verifica e generazione hash SHA256/MD5.



\## 🛠️ Installazione

Configurato per girare in ambiente WSL con accesso a `C:\\Users\\%USERNAME%\\CyberShadow`.
