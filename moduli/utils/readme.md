\# Kali-Python Suite

Terminale protetto per test di rete e web security.



\## Struttura

\- `kali.py`: Core del terminale con autenticazione SHA-256 + Salt.

\- `moduli/sniffing/`: Scanner di rete basato su Scapy.

\- `moduli/vulnshop/`: Applicazione Flask vulnerabile per test SQLi/XSS.



\## Installazione

1\. Installa i requisiti: `pip install scapy flask`

2\. Avvia: `python kali.py`
