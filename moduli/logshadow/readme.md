# Modulo Logshadow

Questo modulo gestisce l'acquisizione log dai dispositivi USB per la suite CyberShadow.

## ⚖️ Licenza Speciale
A differenza del resto della suite, questo specifico modulo è rilasciato sotto licenza **MIT**. 
Puoi riutilizzare liberamente la logica di acquisizione USB contenuta in questa cartella.

## 🛠️ Funzioni
- `cattura_usb(device_id)`: Streaming e salvataggio dati.
