# 🛡️ CyberShadow OS v1.6

CyberShadow OS è una shell interattiva avanzata scritta in Python per scopi di sicurezza informatica e analisi crittografica. Include un sistema di auto-apprendimento per la gestione degli hash e strumenti di network tracking.

## 🚀 Caratteristiche
- **Auto-Learning Hash**: Ogni volta che generi un hash, viene salvato nel database.
- **DeHasing Engine**: Ricerca istantanea di corrispondenze SHA-256 nel database locale.
- **Web Cloner**: Clonazione rapida di pagine web per test di penetration testing.
- **IP Tracker**: Integrazione con moduli di geolocalizzazione IP.
